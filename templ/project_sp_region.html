Notes

The downloaded region files can be plotted using the kappa package command ardplot. For example to overlay the region on an existing file: 

display IMAGE.sdf
ardplot region=REGION.ast

This should also work for the STC-S files.

In the PNG image plot, observations are color-coded as follows:

  • White: new observations.
  • Red: observations in progress, when it is possible to determine that an observation has been observed, otherwise it will appear white until complete.
  • Blue: completed observations.
