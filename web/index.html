OMP Home

These pages describe the JAC Observation management project (OMP). The OMP provides tools and define processes to aid with flexible-scheduled observing at the JCMT and UKIRT. It builds on the JCMT Observation Management Project and the UKIRT ORAC project.

All users:

  • Access the online feedback system for all project information (P.I.s and staff).
    • Observers' version is here

  • Issue a new password for a project

  • Access the fault system

JCMT specific:

  • Primer on using the JCMT Observing Tool (JCMT OT)

UKIRT specific:

  • Documentation for the UKIRT Observing Tool (UKIRT OT)

  • Current UKIRT instructions on using the OMP

Restricted access:

  • Observing Reports for JCMT and UKIRT

  • List projects according to semester, support scientist and country

  • Change project details

  • Display MSBs observed on a specified UT date sorted by project

  • Display and update OMP user details

  • View and edit support contacts for a specific project
